DUDEX64 - Secure Delete

Ensure files are deleted without remains of their contents

This program is able delete files so that none of their remaining content can be found on the disk, which helps prevent file recovery. This is ideal for those who require deleting files so that their sensitive data cannot be found though filesystem analisis.

DISCLAMER: On some systems, like Windows® for example, they may automatically make back-ups of the file in case it is accidently deleted. This program cannot protect your privacy in this situation.
