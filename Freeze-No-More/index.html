DUDEX64 - Freeze No More

Stops windows PCs from freezing.

See the GitHub® Repository for this program

Download the ZIP of the GitHub® Repository for this program

If you have a cheap or underperforming PC that freezes all the time, this program will prevent that. This program automatically ends any other running tasks that are not responding, in order to prevent a lock up.
