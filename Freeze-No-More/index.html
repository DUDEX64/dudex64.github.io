DUDEX64 - Freeze No More

Stops Windows® PCs from freezing.

See the release of this program on GitHub®

Visit Freeze No More's GitHub® Repository




If you have an underperforming PC that freezes all the time, this program will prevent that. This program automatically ends any other running tasks that are not responding, in order to prevent a lock up.
